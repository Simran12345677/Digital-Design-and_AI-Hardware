# Digital Design & RTL Verification Portfolio

This repository contains my digital circuit design and hardware verification projects implemented in Verilog.

## Project Directory

* **[8-Bit ALU](./8-Bit_ALU)** - An 8-bit Arithmetic Logic Unit capable of performing core arithmetic and logical operations, verified with a comprehensive testbench.
* **[Synchronous FIFO](./Synchronous_FIFO)** - A synchronous FIFO memory buffer designed for reliable data queuing under a single clock domain.
* **[Vending Machine FSM](./Vending_Machine_FSM)** - A synchronous Finite State Machine (FSM) controller modeling a vending machine's sequence logic.

## Tools Used
* **HDL:** Verilog
* **Simulation & Verification:** ModelSim / Icarus Verilog / EDA Playground (Jo bhi aapne use kiya ho)
